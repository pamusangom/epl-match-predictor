# Premier League Match Predictor

This is a project that displays data about matches in the premier league, including the teams playing and the final score of the match. A prediction will then be made about future games based on some of the data retrieved.

## Current Features
- Loads data from a JSON file regarding the matches
- Shows the fixtures with the final scores or TBD if the match hasn't yet occured

## Tech Stack
- Python
- JSON
- Git & GitHub

## Status
Currently working on an algorithm that comes up with predictions for the matches!
